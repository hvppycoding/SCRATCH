# SCRATCH

Scratch codes and e.t.c.

## AI Agent Examples

이 저장소에는 다양한 AI 에이전트 구현 예제가 포함되어 있습니다. 에이전트는 도구를 사용하여 파일 시스템 작업, 코드 실행, 데이터 분석 등을 수행할 수 있습니다.

### 주요 에이전트 파일:
- `agent_integration_example.py` - 통합 에이전트 프레임워크
- `openai_agent_with_tools.py` - OpenAI 함수 호출을 사용한 에이전트
- `autogen_vllm_example.py` - AutoGen 멀티 에이전트 시스템
- `file_system_tools.py` - 파일시스템 도구 라이브러리

자세한 내용은 [AGENTS_README.md](./AGENTS_README.md)를 참조하세요.
